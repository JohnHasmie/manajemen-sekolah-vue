Ambil Presensi · Sheet Redesign

The current sheet is a chip picker (Pilih Kelas + Pilih Mapel) with no temporal context — no date, no jam ke-N, no awareness of the teacher's actual schedule. The redesign leads with today's schedule as tappable cards (Jam 1 · 7A · B. Arab · 07.00–07.45 · status pill), pins the live "Sesi sekarang" session at the top, and keeps a manual fallback tab for off-schedule input.

What's changing & why
  • Schedule-driven default. The sheet leads with today's actual teaching slots from teaching_schedules. Each card shows Jam ke-N, time range, subject, class, student count, and a status pill (belum / sudah / sedang). Tap a card → goes straight to the per-student picker for that exact session — no chip-juggling.
  • "Sesi sekarang" pinned banner. Pulse dot + cobalt CTA when the current time falls inside a scheduled slot. One-tap to start the live session, which is the most common case.
  • Sudah-diabsen state. Past sessions show "Sudah · 22 hadir" instead of being unavailable — tapping opens detail/edit, so teachers can fix mistakes without leaving the sheet.
  • Manual mode preserved as a tab. "Atur sendiri" keeps the date/jam/class/subject pickers for off-schedule input (substitute teaching, makeup classes, holidays). The redesign tightens it: tanggal + jam in a row, then class chips, then subject chips filtered by what the teacher actually teaches in that class. A small footer note tells them when the slot was last absent (anti-duplicate).
  • Empty state for non-school days. Sundays/holidays show a friendly "no schedule today" card that nudges to manual mode rather than an empty list.
  • Brand-cobalt header instead of plain white. Matches the rest of the brand surfaces (Presensi list, Kegiatan Kelas detail) — feels like the same app.
