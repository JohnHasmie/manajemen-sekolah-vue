Tahun Ajaran & Pengaturan Umum — Admin Redesign

Five frames covering the cleanup of the "Pengaturan Umum / Tahun Ajaran" surface. Today the menu entry says Tahun Ajaran but the screen only renders Informasi Sekolah, the edit dialog is a hand-rolled AlertDialog, and there is no actual list of academic years. This redesign (a) renames the hub correctly, (b) adds a dedicated Kelola Tahun Ajaran screen with current-year hero + archive list + FAB, (c) replaces both edit dialogs with brand AppBottomSheet, and (d) adds a confirmation dialog for the high-stakes "Aktifkan" action.

  • Filter "Tahun Ajaran" di semua modul beralih ke 2026/2027

New ConfirmationDialog variant. Switching the active year cascades into every other module (filter scope, read-only locks, notifications). Today the action would happen silently with no confirmation. Uses shared ConfirmationDialog with an amber warning icon + bulleted impact list to make consequences explicit. Same dialog reused from Frame D when the toggle is flipped, and from Frame B's overflow menu's "Aktifkan" action.
