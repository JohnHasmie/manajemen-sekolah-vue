Admin · Buku Nilai + Rekap Nilai · redesign

Two problems being solved together. (1) Bug — the admin menu labelled "Rekap Nilai" actually opens the per-teacher Buku Nilai screen (no real Rekap exists for admin). Frames A–C show the proposed fix; D shows how the same KPI pattern applies on teacher / parent.
