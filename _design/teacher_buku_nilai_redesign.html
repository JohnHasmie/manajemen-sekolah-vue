Buku Nilai · Teacher Redesign

Deep redesign of the teacher Buku Nilai (Grade Input) feature, mirroring the same patterns already shipped for Presensi, Kegiatan Kelas, Rekap Nilai, and RPP. Cobalt teacher chrome (BrandPageHeader + KPI overlap), FilterChipGrid selectors everywhere, full-width sheets at 96% viewport for editing, and a clean per-cell score editor with grade-type chips + note. Six frames cover list, grid, and editor flows.

  • Frame A · Overview — class+subject cards with assessment-type chip pills + per-card progress strip + cobalt KPI strip.
  • Frame B · Score grid (card view) — student cards with 4-cell mini scoreboard, tappable per cell.
  • Frame C · Score grid (table view) — frozen-column table, color-coded scores, stacked sticky header.
  • Frame D · Per-cell score editor sheet — big number input, grade-type chips, note field, violet ✦ "Salin dari rerata" suggestion.
  • Frame E · Add assessment header sheet — assessment name + type chips + date + max score, all chip-driven.
  • Frame F · Filter sheet — kelas + mapel + grade-type filter chips, follows the RPP filter sheet shape.
